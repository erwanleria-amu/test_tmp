Description détaillée

Represents a single PHP requirement, e.g. an installed extension. It can be a mandatory requirement or an optional recommendation. There is a special subclass, named PhpIniRequirement, to check a php.ini configuration.

Auteur
Tobias Schultze http://tobion.de
